# Astro MVP / public open source api data ingest

A barebones (ha-ha, see what I did there?) catalogue of dogs, to be extended when inspiration hits me.

Currently it has a filter by first letter, has local caching, nanostore state handling, and Astro.js presentational main structure to significantly improve responsiveness.

### Aim of this project:

Designed merely for practice. Shame the API didn't contain any pictures of dogs. All credits and attributions given in main index.astro page file.
